> Vue-Galleries 是一套专为 vue 框架打造的图片展示组件  

- Gallery:(画廊展示系统-走马灯)  
- Image:(图片展示系统)  
- Preview:(图片放大预览)  
- Waterfall(瀑布流展示)  
---
## 快速开始
### 安装
```console
npm install vue-gallery -S
or
yarn add vue-gallery
```

Vue-Galleries 遵循 [MIT 开源协议](../LICENSE).

###快速开始
```js
import Vue from 'vue'
import VueGalleries from 'vue-galleries'
...
Vue.use(VueGalleries)
...
```
